Basic Commands

!ping

Basic

Test bot response and check latency

Usage: !ping
Response: "Pong! üèì Latency: 45ms"

!help

Basic

Display all available commands with descriptions

Usage: !help
Alternative: !commands

!status

Basic

Show bot status, uptime, and system information

Usage: !status
Shows: Uptime, memory usage, active sessions

!info

Basic

Get detailed chat and group information

Usage: !info
Shows: Chat type, member count, admin list

!sessionid

Basic

Get your unique session ID for support

Usage: !sessionid
Returns: Your unique session identifier

!version

Basic

Check current bot version and updates

Usage: !version
Shows: Current version, last update

Group Management Commands

!tagall [message]

Premium

Tag all group members with a custom message

Usage: !tagall Meeting in 10 minutes!
Alternative: !everyone [message]

!tagallexcept [users] [message]

Premium

Tag all members except specified users

Usage: !tagallexcept @user1 @user2 Team update
Note: Mention users to exclude them

!groupinfo

Basic

Get detailed group information and statistics

Usage: !groupinfo
Shows: Name, description, member count, creation date

!members

Basic

List all group members with their roles

Usage: !members
Alternative: !listmembers

!admins

Basic

List all group administrators

Usage: !admins
Shows: All users with admin privileges

!promote @user

Admin Only

Promote a user to group administrator

Usage: !promote @username
Requires: Bot must be group admin

!demote @user

Admin Only

Remove admin privileges from a user

Usage: !demote @username
Requires: Bot must be group admin

!kick @user

Admin Only

Remove a user from the group

Usage: !kick @username
Warning: This action cannot be undone

!mute @user [time]

Admin Only

Temporarily mute a user in the group

Usage: !mute @username 30m
Time formats: 30s, 5m, 2h, 1d

!screenshot [url]

Usage Tips

Command Prefix

All commands start with ! by default. This can be changed by admins.

Case Sensitivity

Commands are not case-sensitive. !HELP works the same as !help.

Reply Commands

Some commands work by replying to messages (like !sticker).

Permission Levels

Commands are marked as Basic, Premium, or Admin Only based on your subscription.
